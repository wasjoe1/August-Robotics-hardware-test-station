Instructions

0. Plug in CB
1. Click on CB button (Ensure CB is already plugged in else repeat from step 0)
2. Observe readings (Configurations, Data, State, User Information)
3. NG/G displays whether the BRITER has valid readings (This is not available when only VSMD is connected)
4. Click on the required Unit ID (Only if the current Unit ID is wrong/ invalid)
5. Check new readings as needed
6. Plug out the CB USB from the microcontroller to disconnect
* Ensure that the CB is the only component plugged in
* Do not plug in other hardware components while testing          
        
0. 插入CB
1. 点击CB按钮（确保CB已插入，否则从第0步重新开始）
2. 观察读数（配置、数据、状态、用户信息）
3. NG/G显示BRITER是否有有效读数（仅当仅连接VSMD时不可用）

5. 检查新的数据
6. 从微控制器拔出CB的USB以断开连接
